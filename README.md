## What is StratoSport?

StratoSport is a real-time football intelligence platform that analyzes any real completed football match using AI. Type in any match — World Cup Final, Champions League Final, El Clásico — and get a full cinematic tactical breakdown in seconds.

## Features

- ⚽ **Any Football Match** — World Cup, Champions League, Premier League, La Liga, Euros and more
- 🧠 **6 Intelligence Modes** — Analyst, Fan, Strategist, War Room, Scout, Story
- 📊 **Rich Visualizations** — Interactive pitch with real player positions, momentum curve, radar charts, phase metric bars
- 🌌 **9 Animated Backgrounds** — Cosmic, Stadium, Desert, Mountains, Aurora, Rainy Night, Tropical, Blizzard, Volcanic
- 🎨 **10 Visual Themes** — Dark, Neon, Amber, Ice, Light, Sunrise, Forest, Ocean, Rose Gold, Slate
- ⚡ **Powered by Groq AI** — Llama 3.3 70B Versatile, ultra-fast inference
- 🔬 **Alternate Reality Simulator** — Adjust pressing, defensive line, tempo and width to explore what could have happened
- 👁️ **Player Intelligence** — Radar charts, role classification, tactical influence scoring for key players
- 🔑 **Free to use** — Bring your own Groq API key (free at console.groq.com)

## How to Use

1. Get a free API key at [console.groq.com/keys](https://console.groq.com/keys)
2. Open the app, paste your key and click **Save & Use Key**
3. Type any real football match and click **Analyze →**

## Analysis Modes

| Mode | What You Get |
|---|---|
| **Analyst** | Full tactical breakdown — pitch, phases, momentum, lineups, deep analysis |
| **Fan** | The story of the match in plain, engaging language |
| **Strategist** | Formation analysis, pressing maps, transition breakdowns |
| **War Room** | Dense 3-column intelligence grid for serious analysts |
| **Scout** | Player-by-player intelligence with radar charts |
| **Story** | Cinematic narrative of the entire match |
